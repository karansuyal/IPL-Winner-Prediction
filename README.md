## 📌 Project Overview

An end-to-end Machine Learning project that predicts the winner of an IPL match based on teams, venue, and toss details. Built for **Celebal Technologies Campus Placement**.

### 🔍 Live Demo

https://github.com/user-attachments/assets/your-screenshot

## 📊 Key Features

- **ML Model**: Random Forest Classifier
- **Accuracy**: 42.74% (IPL is unpredictable!)
- **Data**: 1,169 IPL matches (2008-2024)
- **Features**: Teams, Venue, Toss Winner, Toss Decision
- **Deployment**: Streamlit Web App
